== Description ==

This plugin will help you to create custom coupon in your WooCommerce site.

== DISCOUNT TYPES AND FEATURES ==

* Site admin can set custom coupon for the customer. It will helps to improve products sale. You can set the coupon as percentage discount or fixed price discount based on your need. This coupon amount will reflect in your cart page.

= Percentage Discount Example =
* Buy laptop from Category Asus and save 10% for Asus laptop bag when apply your custom coupon

= Fixed Discount Example =
* Buy one Belt And Get one Belt with 20 OFF recursively when apply your custom coupon

== Installation ==

Download the plugin in your directory. Upload it in your site add plugin page and active the plugin. Also you can manually copy the plugin folder in your site plugin folder PATH(your_wordpress_path/wp-content/plugins) and you can active the plugin.  

= Minimum Requirements =

* WordPress 4.0.0 or greater
* WooCommerce 3.0.0 or greater
* PHP version 5.6.0 or greater
* MySQL version 5.0 or greater
